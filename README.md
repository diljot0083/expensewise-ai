# ExpenseWise AI 🚀

A modern full-stack expense tracking application that helps users manage, analyze, and optimize their spending with a clean UI and intelligent insights.

Designed to simplify personal finance tracking with real-time insights and a smooth user experience.

## 🌐 Live Demo

https://expensewise-ai.vercel.app/

## ✨ Features

* Add, edit, and delete expenses
* Dashboard with total spending insights
* Category-based filtering
* Weekly and monthly spending analysis
* Interactive charts (category breakdown & trends)
* AI-powered insights (spending summary, top category, smart suggestions)
* Secure authentication with Google OAuth and JWT
* Clean UI with dark mode
* Real-time data handling with backend integration

## 🤖 Enhancements in Progress

* Advanced AI-based expense categorization
* Smarter insights and recommendations
* Improved mobile responsiveness
* Enhanced analytics and reporting

## 🛠 Tech Stack

* Frontend: React + TypeScript + Tailwind CSS
* Backend: Node.js + Express
* Database: MongoDB
* Deployment: Vercel
